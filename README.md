**Sustainable Campus Governance Platform**
      A web-based platform designed to help educational institutions monitor and manage sustainability initiatives such as energy usage, water consumption, waste management, events, and student feedback through a centralized system with role-based access control.

**Features**
**Admin**
Secure login authentication
Add and manage sustainability data
Post and manage sustainability events
View sustainability scores
View student feedback
**
Student**
Secure login authentication
View sustainability scores
View upcoming sustainability events
Submit eco-feedback

**Technology Stack**

Frontend:
HTML
CSS
JavaScript
React.js

Backend:
